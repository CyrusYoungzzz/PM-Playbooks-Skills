# Product 方法论

一套面向产品决策的命令式工具集。把不同产品大牛的方法论拆成可触发的 `/command`，按场景调用，避免方法论散落在博客和书里没人看。

> 如何扩展、设计原则、使用示例：见 [CONTRIBUTING.md](./CONTRIBUTING.md)。

## 当前收录

| 来源 | 方法论 | 沉淀文件 |
|------|--------|---------|
| 课代表立正（B 站 UP） | 真伪需求 / Product Sense | `references/kedaibiao-real-vs-fake.md` |
| 前小米 李创奇（前小米生态链高级产品总监；主导小米电视 / 米家投影 / 巨能写 / AI 眼镜） | 爆品方法论（预期违背 + 长板理论 + 爆品三方法论 + 回声效应） | `references/lichuangqi-baopin.md` |

原始素材分别在 `kedaibiao-raw/` 和 `lichuangqi-raw/` 下，蒸馏过程不丢。

## 命令清单（按使用场景）

| 命令 | 适用场景 | 调用的方法论 |
|------|---------|-------------|
| `/真伪需求分析` | 拿到一个新需求 / 新功能想法，先判断是真需求还是幻觉 | 课代表 · 真伪需求 |
| `/爆品立项` | 评估一个产品要不要做、值不值得做成爆品 | 李创奇 · 长板理论 + 爆品三方法论 |
| `/超预期设计` | 给一个产品/功能设计能让用户传播的体验 | 李创奇 · 预期违背四层 + 锚点四来源 |
| `/预期诊断` | 用户说"还不错"但没人传播 → 定位是哪种"果然如此"在作祟 | 李创奇 · 果然如此四对齐 + 贬值信号 |
| `/品类立项` | 一个新硬件品类要不要做 / 何时入局 | 李创奇 · 立项三问 + 销量反推 |
| `/产品老化` | 产品迭代到第 N 代变平庸 → 升层 / 降维 / 反预期 | 李创奇 · 超预期贬值与应对 |
| `/回声效应` | 设计品牌矩阵的节奏，让新品借上前一次的预期势能 | 李创奇 · 第四层 + 变奏不变调 |
